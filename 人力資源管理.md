# HR人力资源角色功能模块说明
| 功能模块       | 具体功能                                                                 | 功能说明                                                                 |
|----------------|--------------------------------------------------------------------------|--------------------------------------------------------------------------|
| 基础信息管理   | 员工考勤档案管理                                                         | 为每位员工建立独立考勤档案，关联员工部门/职位/工龄，同步管理员更新的员工基础信息，档案包含员工入职至今所有考勤、请假记录 |
|                | 假期额度全局监控                                                         | 实时监控所有员工各类假期（事假/婚假/产假/年假）的剩余额度，对额度不足的申请进行预警 |
| 考勤业务管理   | 跨部门请假审批                                                           | 处理员工跨部门工作的请假申请（部门经理初审后），进行终审并记录审批意见     |
|                | 考勤规则落地监督                                                         | 监督各部门执行管理员设置的上班时间、假期时长等规则，对违规执行情况反馈给管理员 |
| 考勤数据管理   | 全公司考勤数据汇总                                                       | 按日/周/月汇总各部门的打卡、请假、迟到/早退/旷工数据，生成全公司考勤台账   |
|                | 多维度考勤数据查询/排序                                                 | 支持按**部门/职位/考勤类型/时间**进行单一/组合/模糊查询，可对查询结果按多属性排序（如按部门+迟到次数降序） |
|                | 全公司考勤数据统计分析                                                   | 1. 单一属性统计：全公司迟到/早退/旷工总人数、各类假期总申请数等<br/>2. 多属性统计：各部门+各职位的年假使用平均时长、各部门月度旷工率等<br/>3. 预设统计：按企业常规需求生成月度/季度/年度考勤核心指标统计<br/>4. 条件统计：按自定义条件统计（如统计工龄5年以上员工的年假使用情况、统计某季度全公司事假超3天的员工数） |
| 考勤异常处理   | 全公司考勤异常审核                                                       | 审核各部门提交的迟到/早退/旷工/漏打卡等异常说明，判定异常是否有效，有效则标注销除，无效则计入考勤异常记录 |
|                | 考勤异常记录归档                                                         | 对所有无效考勤异常进行归档，关联员工个人考勤档案，作为员工绩效考核、薪酬核算的依据 |
|                | 考勤异常预警                                                             | 对员工单次异常、月度累计多次异常（如月度迟到超3次）进行系统预警，提醒对应部门经理及员工 |
| 考勤报表管理   | 标准化考勤报表生成                                                       | 按企业人力资源管理要求，生成**部门考勤报表、员工个人考勤报表、全公司考勤汇总报表**，报表包含核心考勤指标（打卡率、异常率、假期使用率等） |
|                | 考勤报表下发/归档                                                       | 将部门考勤报表下发给对应部门经理，将全公司考勤报表及员工个人考勤档案归档存储，支持按时间查询调阅 |
|                | 考勤报表对接薪酬系统（拓展功能）| 将考勤异常数据（如旷工、迟到）按企业规则转换为薪酬扣减依据，生成薪酬核算考勤明细表，供薪资管理系统调用 |
| 系统维护       | 个人密码维护                                                             | 仅可修改自身账号密码，无其他系统维护权限（数据备份/恢复、密码重置等权限仍归管理员） |
| 其他拓展功能   | 考勤制度公示                                                             | 在系统内公示企业考勤管理制度、假期规则，方便所有员工查询                   |
|                | 考勤数据导出                                                             | 支持将考勤台账、报表、异常记录等数据导出为文本文件，方便线下存档和分析     |
